## 📝 Sobre o Projeto
O **AcessMap** é um aplicativo mobile projetado para transformar a maneira como pessoas com deficiência ou mobilidade reduzida interagem com o espaço urbano. O projeto foca no mapeamento colaborativo de pontos estratégicos, permitindo que a comunidade identifique desde rampas de acesso, rotas e até estabelecimentos comerciais com infraestrutura adequada.

### Principais Funcionalidades:
- **Login e cadastro:** Usuários realizam um login e inserção de dados para acessar o aplicativo.
- **Mapeamento Colaborativo:** Marcação de pontos com categorias como Rampas, Vagas PCD, Obstáculos e Perigos.
- **Filtro de Localização:** O sistema identifica o tipo de terreno (rua, calçada ou prédio) e valida se o marcador é compatível com o local.
- **Feedback por Popups:** Avisos claros em tela que informam o usuário sobre a validade do local e a obrigatoriedade de fotos e descrições, tornando o aplicativo intuitivo.
- **Localização do Usuário:** O aplicativo mostra a localização atual do usuário (requer permissão).

## 🛠️ Ferramentas e Tecnologias
- **Framework:** [Flutter](https://flutter.dev/) (Desenvolvimento multiplataforma).
- **Linguagem:** Dart.
- **Mapas:** [Mapbox Maps SDK for Flutter](https://docs.mapbox.com/flutter/maps/guides/).
- **Hardware de Teste:** Execução e debug otimizados para dispositivos móveis como o **Redmi Note 10**.
